# Nothing

A study in absence. A monochrome, over-engineered tribute to saying nothing — beautifully.

Built with vanilla HTML, CSS and JS. No frameworks. No dependencies. No point.

## Features

- Custom lerp-followed cursor with magnetic interactive elements
- Canvas particle field with cursor-reactive glow and proximity lines
- Per-character text reveals and text-scramble effects
- 3D tilt object showcase with layered rings + spec sheet
- Horizontal-scroll process section (wheel-trapped)
- Animated counters, marquee, sticky section heads
- Grain overlay, blur navigation, live clock
- Fully responsive, respects `prefers-reduced-motion`

## Run locally

Open `index.html` in any modern browser. That's it.
